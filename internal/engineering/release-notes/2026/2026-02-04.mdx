## Breaking Changes

<Warning>
  **API Response Format Change**

  The `/v1/documents` endpoint now returns paginated results by default.

  **Migration:**
  ```bash
  # Old format (deprecated)
  GET /v1/documents

  # New format
  GET /v1/documents?page=1&per_page=50
  ```

  Clients should update to handle pagination or explicitly request all results with `per_page=-1`.
</Warning>

## Deployment Notes

### Pre-Deployment

<Steps>
  <Step title="Database Migration">
    ```bash
    qumis-cli rails:run -e prod -c "bundle exec rake db:migrate"
    ```
  </Step>
  <Step title="Clear Cache">
    ```bash
    qumis-cli rails:run -e prod -c "bundle exec rake cache:clear"
    ```
  </Step>
</Steps>

Standard blue-green deployment via GitHub Actions. No manual intervention required.

### Post-Deployment

<Steps>
