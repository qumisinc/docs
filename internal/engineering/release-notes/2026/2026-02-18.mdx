## Features

### Landing Page CRO Overhaul

Comprehensive conversion rate optimization pass on the landing page to improve engagement and signup rates.

- **FAQ section** added and centered on the landing page
- **Sticky CTA button** — persistent call-to-action visible while scrolling
- **SOC 2 trust badges** added to the hero section and trial signup card
- **Typography improvements** — increased body font sizes for readability
- **Visual polish** — refined CTA button styling, gradient effects, fixed navigation border flash on page load
- **Hero and animation fixes** — smoother transitions and layout refinements

**Linear:** [ENG-1647](https://linear.app/qumis/issue/ENG-1647)

### Terms of Service

Users signing up now agree to both the Privacy Policy and a new Terms of Service — accessible via a modal in the signup flow.

- New `LegalDocumentModal` component for rendering legal markdown content
- Full Terms of Service content authored in markdown
- Signup agreement text updated to reference both documents

**Linear:** [ENG-1661](https://linear.app/qumis/issue/ENG-1661)

### OG Share Image

Branded Open Graph image for social media and messaging link previews, ensuring Qumis links look professional when shared.

- New `qumis-og-image.png` and `@2x` retina variant
- OG meta tags added to `index.html`
- Custom HTML template for image generation

**Linear:** [ENG-1647](https://linear.app/qumis/issue/ENG-1647)
