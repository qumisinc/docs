### Categories

| Category | Description | Icon |
|----------|-------------|------|

## Creating Release Notes

<Steps>
  <Step title="Copy the template">
    Copy the latest release file and rename with today's date: `YYYY-MM-DD.mdx`
  </Step>
  <Step title="Update the content">
    Fill in deployment details, features, bug fixes, and any breaking changes
  </Step>
  <Step title="Update navigation">
    Add the new file to `docs.json` under the appropriate year group
  </Step>
  <Step title="Update this hub">
    Add a card to Recent Releases and update the Archive table
  </Step>
</Steps>
