A quick-reference guide for translating common insurance terms between US English and Australian English.

## General insurance terms

| US English | Australian English |
| --- | --- |
| Insurance policy | Insurance policy (same) |
| Premium | Premium (same) |
| Deductible | Excess |
| Copay / Copayment | Gap payment |
| Claim | Claim (same) |
| Coverage | Cover |
| Policyholder | Policyholder / Policy owner |
| Beneficiary | Beneficiary (same) |
| Underwriting | Underwriting (same) |
| Endorsement | Endorsement / Policy amendment |
| Rider | Benefit extension |
| Exclusion | Exclusion (same) |
| Liability | Liability (same) |

| US English | Australian English |
| --- | --- |

| Homeowners insurance | Home and contents insurance |
| Auto insurance | Motor vehicle insurance / Car insurance |
| Collision coverage | Own damage cover |
| Comprehensive coverage | Comprehensive cover |
| Liability coverage | Third party liability cover |
| Personal injury protection (PIP) | Compulsory Third Party (CTP) |
| Uninsured motorist coverage | Uninsured motorist extension |
| Actual cash value (ACV) | Market value |
| Replacement cost | Replacement value / New for old |
