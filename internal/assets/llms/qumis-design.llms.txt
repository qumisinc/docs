# Qumis Style Guide

Brand and design system reference for Qumis, an AI platform for insurance professionals.

## Brand Overview

- **Brand Name**: Qumis
- **Version**: 2.4
- **Tagline**: Your co-pilot trained specifically for insurance
- **Secondary Message**: Expert AI built for the top minds in insurance

## Visual Identity

### Wordmark
- **Primary Typeface**: Clean, minimal sans-serif (Inter or similar)
- **Style**: Lowercase "qumis" with consistent letter spacing
- **Usage**: Primary brand expression in dark text on light backgrounds

### Icon
- Circular "Q" icon used in navigation and branding elements


## Color System

### Primary Interface Colors
Based on actual UI implementation:

- **Background**: #FFFFFF (Pure white for main content areas and cards)
- **Sidebar Background**: #F7F7F6 (Neutral light gray for navigation sidebar)
- **Modal Background**: #F5F5F4 (Light gray for modal overlays)
- **Primary Text**: #1A1A1A (Near-black for headings and primary content)
- **Secondary Text**: #4A4A4A (Dark gray for body text and descriptions)
- **Muted Text**: #737373 (Medium gray for metadata, timestamps, and helper text)
- **Lighter Text**: #9B9B9B (Light gray for less prominent information)

### Interactive Elements
- **Primary Button**: #1A1A1A (Near-black background with white text)
- **Primary Button Hover**: #2D2D2D (Slightly lighter on hover)
- **Secondary Button**: White background with #D4D4D4 border
- **Links/Actions**: #1A1A1A with underline or hover effects
- **Icon Color**: #4A4A4A (Consistent with secondary text)

### Structural Elements
- **Borders**: #E5E5E5 (Light gray for card borders and dividers)
- **Subtle Borders**: #F0F0F0 (Very light gray for minimal separation)
- **Table Borders**: #EBEBEB (Subtle lines for table structure)
- **Card Shadows**: rgba(0, 0, 0, 0.05) (Very subtle shadow for depth)

### State Colors
- **Hover Background**: #F9F9F9 (Subtle gray for hover states on cards/rows)
- **Selected/Active**: #F0F0F0 (Light gray background for active navigation items)
- **Focus Outline**: #1A1A1A (Dark outline for accessibility)

### CSS Design Tokens

```css
:root {
  /* Backgrounds */
  --background: #FFFFFF;
  --background-secondary: #F7F7F6;
  --background-modal: #F5F5F4;
  --background-hover: #F9F9F9;
  --background-active: #F0F0F0;
  
  /* Text Colors */
  --foreground: #1A1A1A;
  --foreground-secondary: #4A4A4A;
  --foreground-muted: #737373;
  --foreground-light: #9B9B9B;
  
  /* Interactive */
  --primary: #1A1A1A;
  --primary-hover: #2D2D2D;
  --primary-foreground: #FFFFFF;
  
  /* Borders & Structure */
  --border: #E5E5E5;
  --border-subtle: #F0F0F0;
  --border-table: #EBEBEB;
  
  /* Layout */
  --radius: 6px;
  --shadow-sm: 0 1px 3px rgba(0, 0, 0, 0.05);
  --shadow-md: 0 4px 6px rgba(0, 0, 0, 0.07);
}
```

## Layout System

### Navigation
- **Left Sidebar**: 
  - Width: ~240px
  - Background: #F7F7F6
  - Items: 40px height with 12px padding
  - Icons: 20px size, #4A4A4A color
  - Active state: Subtle #F0F0F0 background
  - Typography: 14px, medium weight

### Content Areas
- **Main Content**: 
  - Pure white (#FFFFFF) background
  - Maximum width for readability
  - Padding: 24-32px

- **Cards/Tiles**: 
  - White background
  - 1px border (#E5E5E5)
  - Border radius: 6px
  - Padding: 16-20px
  - Hover: Subtle shadow increase

- **Modal Dialogs**: 
  - White card on semi-transparent dark overlay
  - Max width: 500-600px
  - Rounded corners (8px)
  - Padding: 24-32px

### Data Display
- **Tables**: 
  - Header: Light gray background (#F9F9F9)
  - Borders: #EBEBEB
  - Row hover: #F9F9F9
  - Cell padding: 12px 16px

- **Report Layout**: 
  - Numbered sections with clear hierarchy
  - Citation superscripts in smaller, muted text
  - Line height: 1.6 for readability

## Typography

- **Font Family**: Inter, system-ui, -apple-system, sans-serif
- **Font Sizes**:
  - Heading 1: 24-28px
  - Heading 2: 20px
  - Body: 14px
  - Small/Meta: 12px
- **Font Weights**:
  - Regular: 400 (body text)
  - Medium: 500 (navigation, labels)
  - Semibold: 600 (headings, buttons)
- **Line Heights**:
  - Headings: 1.3
  - Body: 1.6
  - Compact (tables, navigation): 1.4

## Interactive Components

### Buttons
- **Primary**: 
  - Background: #1A1A1A
  - Text: #FFFFFF
  - Height: 36-40px
  - Padding: 0 16px
  - Border radius: 6px
  - Font weight: 500

- **Secondary**: 
  - Background: #FFFFFF
  - Border: 1px solid #D4D4D4
  - Text: #1A1A1A
  - Same dimensions as primary

### Form Elements
- **Inputs/Search**: 
  - Border: 1px solid #E5E5E5
  - Background: #FFFFFF
  - Height: 36px
  - Focus border: #1A1A1A
  - Placeholder: #9B9B9B

### Icons
- **Style**: Outline/line icons
- **Size**: 20px standard, 16px small, 24px large
- **Color**: #4A4A4A default, #1A1A1A on hover/active
- **Stroke Width**: 1.5-2px

## Usage Guidelines

- Maintain minimal color usage - the interface is intentionally monochromatic
- Use subtle shadows and borders for hierarchy, not color
- Keep contrast ratios above 4.5:1 for normal text, 3:1 for large text
- Apply consistent 8px spacing grid
- Use hover states sparingly and subtly
- Reserve pure black (#000000) for special emphasis only
- Ensure all interactive elements have clear visual feedback
